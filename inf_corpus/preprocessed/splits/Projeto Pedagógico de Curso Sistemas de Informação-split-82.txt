Métodos de projeto de interação. Projeto visual (cores, ícones,
fontes e outros). Tempo de resposta e retroalimentação. Dispositivos de interação. Métodos de
avaliação de interfaces: avaliação heurística, abordagens para testes realizados com apoio de
usuários, técnicas de testes para páginas Web, entre outros. Visão geral de ferramentas de
desenvolvimento de interfaces humano-computador.
